<h1>A simple x86_32 Emulator writed on C</h1>

---

### What is the planned?
- A full realmode support
- All 8086 opcodes
- IVT table
- In future support protected mode and more x86_32 opcodes

---

### What is currently?
- Registers
- Prefix Support
- Fetch-Decode cycle
- Read and Write to memory / from memory
- Based Opcodes
