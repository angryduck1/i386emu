<h1>x86_32 Emulator written on C</h1>

---

### What is the planned?
- A full realmode and 8086 opcodes
- IVT table
- In future support protected mode and more x86_32 opcodes

---

### What is currently?
- All based registers
- ModR/M Support
- Active developing opcodes
- Support Prefixes
